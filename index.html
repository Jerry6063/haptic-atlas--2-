// Treating the city as an information field re-routed from vision and sound into the skin.

01 // PROJECT DESCRIPTION

The Commute as Interface

Haptic Atlas is a research-through-design project that follows a daily commute from Long Island City to Columbia University. It investigates how navigation information—traffic signals, urban landmarks, spatial transitions—is unevenly distributed.

The project proposes a wearable haptic belt to translate this visual field into structured tactile patterns.

  • Explores gaps in urban accessibility along a real route.
  • Prototypes a belt-based haptic navigation system.
  • Frames the city as a multi-sensory interface.
